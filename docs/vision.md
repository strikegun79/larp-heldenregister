# LARP Heldenregister

Webanwendung für Kinder- und Jugend-LARP.

Ziele:
- Charakterverwaltung
- Abenteuerhistorie
- Fertigkeiten-Baum pro Klasse
- Gruppenverwaltung
- Veranstaltungsverwaltung
- öffentliche Heldenansicht ohne Realname, Heldensuche per 6-stelligen Code, den jedes Kind in Real erhält und in seinem Heldenprofil sehen kann.


Technik:
- Laravel
- MySQL
- Blade
- GitHub
- Fomantic-UI
