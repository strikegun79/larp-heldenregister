# LARP Heldenregister

Webanwendung für Kinder- und Jugend-LARP.

Ziele:
- Charakterverwaltung
- Abenteuerhistorie
- Fertigkeiten-Baum pro Klasse
- Gruppenverwaltung
- Veranstaltungsverwaltung
- öffentliche Heldenansicht ohne Realname, Heldensuche per 6-stelligen Code, den jedes Kind in Real erhält und in seinem Heldenprofil sehen kann.

Ablauf neue Spieler und Helden:
	- Eltern registrieren sich
	-- Eltern legen ihre Kinder/Spieler an
	- Der Bürokrat erstellt die Helden (nicht die Spieler oder Eltern)
		-- Der Bürokrat weist den Helden dem Spieler zu.
	- Eltern/Spieler wählen ihren aktiven Helden zum Abenteuer, der die EXP erhält (sollte nicht die Regel sein)
	- Der Bürokrat schaltet die Fertigkeiten frei die sie sich verdient haben


Technik:
- Laravel
- MySQL
- Blade
- GitHub
- Fomantic-UI
